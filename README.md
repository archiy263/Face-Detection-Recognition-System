# Face Detection & Recognition System

This project performs real-time face detection and recognition using Python, OpenCV, and the face_recognition library.

## Features
- Capture new faces
- Train face encodings
- Real-time recognition from webcam
- Supports multiple faces

## Setup

Activate environment:
conda activate faceai

Install requirements:
pip install -r requirements.txt

## Run

1. Add faces:
python capture_faces.py

2. Train:
python train_encodings.py

3. Recognize:
python recognize.py
